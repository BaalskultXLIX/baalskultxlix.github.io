QEMU

Ich brauche um QEMU zu starten qemu-full, passt für internet und edk2-ovmf für UEFI . Ich starte QEMU über mein USB stick ----> ( ... /dev/sda ... ) <----

Arch Linux: sudo pacman -S qemu-full passt edk2-ovmf

Ich muss zuerst ein imge file erstellen mit dem command "qemu-img create -f raw image_file 4G"

Ich muss als zweites passt starten um internet zu habe mit dem command "passt -f" (Mann muss es gleichzeitig mit QEMU starten.).

Hier sind meine Arguments "sudo qemu-system-x86_64 -enable-kvm -machine q35 -device intel-iommu -cpu host -cdrom /dev/sda -boot order=d -drive if=pflash,file=linux,format=raw,file=/usr/share/edk2/x64/OVMF.4m.fd -m 4G -device virtio-net-pci,netdev=s \ -netdev stream,id=s,server=off,addr.type=unix,addr.path=/tmp/passt_1.socket"
